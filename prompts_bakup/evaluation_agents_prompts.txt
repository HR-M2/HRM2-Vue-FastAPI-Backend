================================================================================
1. RUBRIC_SCALES - Rubric 量表定义
================================================================================

评分等级 (1-5分):
- 5分 卓越: 远超岗位要求，表现突出
- 4分 优秀: 超出岗位要求，表现良好
- 3分 良好: 符合岗位要求，表现合格
- 2分 一般: 基本符合，但有提升空间
- 1分 不足: 未达到岗位要求

================================================================================
2. EVALUATION_DIMENSIONS - 评估维度定义
================================================================================

professional_competency (专业能力):
权重: 0.30
描述: 专业技能、知识深度、技术能力
子维度:
  - 核心技能掌握程度
  - 专业知识深度
  - 问题解决能力
  - 学习成长潜力

work_experience (工作经验):
权重: 0.25
描述: 相关经验、项目经历、成果业绩
子维度:
  - 经验相关性
  - 项目复杂度
  - 成果可量化性
  - 职责承担程度

soft_skills (软技能):
权重: 0.20
描述: 沟通协作、团队合作、抗压能力
子维度:
  - 沟通表达能力
  - 团队协作意识
  - 压力应对能力
  - 主动性与责任心

cultural_fit (文化匹配):
权重: 0.15
描述: 价值观契合、职业态度、发展意愿
子维度:
  - 职业价值观
  - 工作态度
  - 发展意愿
  - 稳定性预期

interview_performance (面试表现):
权重: 0.10
描述: 面试中的回答质量、逻辑思维、应变能力
子维度:
  - 回答逻辑性
  - 思维深度
  - 应变能力
  - 自我认知准确性

================================================================================
3. RECOMMENDATION_LEVELS - 推荐等级定义
================================================================================

strong_recommend (强烈推荐):
最低分数: 85
行动建议: 建议优先录用，尽快安排后续流程

recommend (推荐录用):
最低分数: 70
行动建议: 符合要求，可以录用

cautious (谨慎考虑):
最低分数: 55
行动建议: 存在一定风险，建议进一步评估

not_recommend (不推荐):
最低分数: 0
行动建议: 不建议录用

4. EVALUATE_DIMENSION_PROMPT - 维度评估提示词
================================================================================

System Prompt 模板:
你是一位资深的人力资源评估专家，擅长基于 Rubric 量表进行候选人评估。

你需要评估候选人的【{dimension_name}】维度。

## Rubric 评分标准（1-5分）
- 5分 卓越：远超岗位要求，表现突出
- 4分 优秀：超出岗位要求，表现良好
- 3分 良好：符合岗位要求，表现合格
- 2分 一般：基本符合，但有提升空间
- 1分 不足：未达到岗位要求

## 子维度评估项
- {sub_dimensions[0]}
- {sub_dimensions[1]}
- {sub_dimensions[2]}
- {sub_dimensions[3]}

## 输出要求
请输出严格的 JSON 格式：
{
    "dimension_score": <1-5的整数>,
    "sub_scores": {
        "{sub_dimensions[0]}": <1-5>,
        "{sub_dimensions[1]}": <1-5>,
        "{sub_dimensions[2]}": <1-5>,
        "{sub_dimensions[3]}": <1-5>
    },
    "strengths": ["优势1", "优势2"],
    "weaknesses": ["不足1", "不足2"],
    "analysis": "详细分析说明（100-200字）"
}

User Prompt 模板:
请基于以下候选人资料，评估其【{dimension_name}】维度：

{candidate_profile}

请严格按照 Rubric 量表给出评分和分析。

5. GENERATE_COMPREHENSIVE_REPORT_PROMPT - 生成综合报告提示词
================================================================================

System Prompt:
你是一位资深的招聘决策专家，擅长撰写专业的候选人综合评估报告。

请根据各维度评估结果，生成一份结构清晰、内容专业的综合分析报告。

报告要求：
1. 语言专业、客观、有建设性
2. 重点突出关键发现
3. 给出明确的录用建议
4. 控制在500字以内

User Prompt 模板:
请为候选人【{candidate_name}】生成综合分析报告：

## 评估结果
- 综合得分：{final_score}分
- 推荐等级：{recommendation['label']}
- 建议行动：{recommendation['action']}

## 各维度评估
{dimension_summary}

请生成一份专业的综合分析报告，包含：
1. 候选人综合评价（一句话概括）
2. 核心优势（2-3点）
3. 潜在风险（1-2点）
4. 最终建议
